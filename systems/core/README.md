## 📐 Padrões de Nomenclatura (Style Guide)

### 📁 Arquivos e Pastas
*   **Tudo em `snake_case`** (minúsculo com underline).
*   **Pastas sempre no plural** (ex: `scenes/`, `animations/`, `projectiles/`). # Deixa no singular se dentro da pasta for um único elemento (ex: "player/")
*   **Exemplo:** `bullet_thompson.tscn`, `bullet_thompson.gd`.

### 💻 No Código (GDScript)
*   **`PascalCase`**: Apenas para nomes de classes (`class_name PlayerCharacter`).
*   **`SCREAMING_SNAKE_CASE`**: Apenas para constantes (`const MAX_SPEED = 300`).
*   **`snake_case`**: Para todo o resto (variáveis, funções, sinais e enums).
